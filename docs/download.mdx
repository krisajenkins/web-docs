# Download

<Boxes>
<Box title="Toit CLI">

Download the version of the Toit CLI that matches your current operating system

Choose [Windows](https://archive.toit.io/cli/latest/windows/installer), [macOS](https://archive.toit.io/cli/latest/darwin) or [Linux](https://archive.toit.io/cli/latest/linux)

</Box> 

<Box title="Toit Examples">

Download the Toit examples for the latest SDK. Choose [ZIP](https://archive.toit.io/sdk/latest/examples/windows) or [TGZ](https://archive.toit.io/sdk/latest/examples/linux)

</Box>

<Box title="Toit SDK Library">

Download the Toit library for the latest SDK. Choose [ZIP](https://archive.toit.io/sdk/latest/lib/windows) or [TGZ](https://archive.toit.io/sdk/latest/lib/linux)

</Box>

<Box title="Toit extension for VSCode">

Download the Toit language extension for Visual Studio Code from [here](https://archive.toit.io/vscode/v1.0.0)

</Box>
</Boxes>
